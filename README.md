Дана програма розроблена для створення та математичної обробки числових двовимірних масивів (матриць) з метою реалізації базових алгоритмів роботи зі структурами даних. Додаток має графічний інтерфейс із двома візуальними таблицями та трьома кнопками управління, кожна з яких відповідає за окремий етап обробки. На першому етапі програма генерує дві незалежні числові матриці розміром 5х5, автоматично заповнюючи їх псевдовипадковими цілими числами в діапазоні від -50 до 50 (що гарантує наявність як додатних, так і від'ємних значень для перевірки умов), після чого одразу відображає їх на екрані. Далі реалізовано функціонал підрахунку середнього арифметичного: алгоритм виконує послідовний обхід усіх елементів в таблиці , відфільтровує лише строго додатні числа, підраховує їхню загальну суму та кількість, а потім обчислює середнє значення і виводить результат у текстову мітку. Третім етапом є структурна зміна масивів, а саме — сортування парних стовпців за спаданням. Програма викопіює дані з цих стовпців, застосовує алгоритм сортування від найбільшого значення до найменшого і перезаписує відсортовані числа назад на їхні місця у таблицях. Загалом, цей додаток є практичним інструментом, який демонструє навички програмування мовою C++, показуючи принципи роботи з виділенням пам'яті, вкладеними циклами, умовними конструкціями та взаємодією з компонентами Windows Forms для швидкої автоматизації математичних обчислень.
